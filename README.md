# 🛸 SOMARS Points Tracker

A drone & aerospace themed points-tracking web app for the SOMARS club.

## Features
- **Home Base** — total club points, live leaderboard, award milestone previews
- **Crew Roster** — 15 member cards with individual point totals and current rank
- **Awards Bay** — full list of all award tiers with progress bars
- **Point buttons** — +1, +5, +10, +25, +50, +100, +250, +500, +1,000, +2,500, +5,000, +10,000, custom amount, and deduct options
- **Rename members** — click any member card → edit name field → Save
- **Persistent storage** — all data saved in `localStorage` (no server needed)

## Running Locally
Just open `index.html` in any browser — no build step needed.

## Deploying to GitHub Pages
1. Push this repo to GitHub (already named `somars-points-tracker` is fine).
2. Go to **Settings → Pages**.
3. Set **Source** to `Deploy from a branch` → `main` → `/ (root)`.
4. Your site will be live at `https://slugbotics.github.io/somars-points-tracker/`

> ℹ️ You do **not** need to rename the repo to `.io` — that's only required for a root-level user/org page.
